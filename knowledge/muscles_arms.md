# 上肢の筋肉 — 解剖学的参照

## 肩部

### 三角筋 Deltoid
- **起始**:
  - 前部: 鎖骨外側1/3
  - 中部: 肩峰
  - 後部: 肩甲棘
- **停止**: 上腕骨三角筋粗面
- **繊維方向**:
  - 前部: 斜め後下方
  - 中部: 垂直（直下）— 多羽状
  - 後部: 斜め前下方
- **形状**: 逆三角形。肩全体を覆う丸みのある筋
- **羽状角**: 中部は多羽状（pinnate）構造で特に複雑
- **3Dモデル判定ポイント**:
  - 3部で繊維方向が異なること
  - 肩の丸みを正確に表現しているか
  - 中部の多羽状を表現できているか

---

## 上腕（前面）

### 上腕二頭筋 Biceps Brachii
- **起始**: 長頭（関節上結節）・短頭（烏口突起）
- **停止**: 橈骨粗面・腱膜
- **繊維方向**: ほぼ垂直（縦方向）
- **形状**: 2頭が合流した紡錘状。前腕回外位で最も膨隆
- **3Dモデル判定ポイント**: 縦方向の繊維、2頭の合流が見えるか

### 上腕筋 Brachialis
- **起始**: 上腕骨前面下部1/2
- **停止**: 尺骨粗面
- **繊維方向**: 垂直
- **形状**: 二頭筋の深層にあるが外側から一部見える

---

## 上腕（後面）

### 上腕三頭筋 Triceps Brachii
- **起始**:
  - 長頭: 肩甲骨関節下結節
  - 外側頭: 上腕骨後面（橈骨神経溝上部）
  - 内側頭: 上腕骨後面（橈骨神経溝下部）
- **停止**: 肘頭
- **繊維方向**: 垂直（各頭から肘頭へ）
- **形状**: 馬蹄形。長頭が内側、外側頭が外側から見える
- **3Dモデル判定ポイント**: 3頭が識別できるか、肘頭への収束

## 筋肉の階層・接続関係（エコルシェモデル配置用）

### 浅層→深層の順（表面から見える優先順位）

#### 肩部

| 深度 | 筋肉 | 備考 |
|---|---|---|
| 最浅層 | 三角筋（前部・中部・後部） | 肩全体を覆う。ほぼすべての肩筋の上に位置 |
| 深層（三角筋下） | 上腕二頭筋長頭（腱部） | 三角筋前縁の下に潜り込む |
| 深層（三角筋下） | 棘上筋・棘下筋・小円筋（ローテーターカフ） | 三角筋の直下 |
| 深層（腋窩部） | 烏口腕筋 | 三角筋前部・大胸筋に覆われる |

#### 上腕前面

| 深度 | 筋肉 | 備考 |
|---|---|---|
| 浅層 | 上腕二頭筋（長頭・短頭） | 上腕前面の最表層 |
| 深層 | 上腕筋 | 二頭筋ほぼ全域の深層。外側から一部覗く |
| 深層・内側 | 烏口腕筋 | 上腕近位1/3の内側、二頭筋短頭の深層 |

#### 上腕後面

| 深度 | 筋肉 | 備考 |
|---|---|---|
| 浅層（外側） | 上腕三頭筋外側頭 | 後外側から最も目立つ |
| 浅層（内側） | 上腕三頭筋長頭 | 後内側・腋窩後壁を形成 |
| 深層 | 上腕三頭筋内側頭 | 長頭・外側頭の深層。肘付近でのみ露出 |

#### 前腕前面（屈筋側）橈側→尺側

| 深度 | 筋肉 | 備考 |
|---|---|---|
| 浅層・橈側 | 腕橈骨筋 | 最外側。肘から際立つ |
| 浅層 | 橈側手根屈筋（FCR） | 腕橈骨筋の尺側隣接 |
| 浅層・中央 | 長掌筋 | FCRの尺側（細い腱で目立ちにくい） |
| 浅層・尺側 | 尺側手根屈筋（FCU） | 最内側の浅層 |
| 浅層・斜走 | 円回内筋 | 肘内側→橈骨中央へ45°斜走。他屈筋の浅層 |
| 深層 | 浅指屈筋 | 上記4筋の深層 |

#### 前腕後面（伸筋側）橈側→尺側

| 深度 | 筋肉 | 備考 |
|---|---|---|
| 浅層・橈側 | 長橈側手根伸筋（ECRL） | 外側最浅層。腕橈骨筋に隣接 |
| 浅層 | 短橈側手根伸筋（ECRB） | ECRLの尺側に隣接 |
| 浅層・中央〜尺側 | 総指伸筋（EDC） | 後面の最大面積を占める |
| 浅層・尺側 | 尺側手根伸筋（ECU） | 最内側の浅層 |
| 深層・近位 | 回外筋 | 前腕近位1/3の深層 |
| 深層・遠位 | 長母指外転筋 | ECRBとEDCの間から表面に斜めに出る |

### 視覚的に「つながる」境界線

| 境界 | 説明 |
|---|---|
| 三角筋前部⇔大胸筋鎖骨部 | **三角胸筋溝**（deltopectoral groove）が鎖骨下から腋窩への縦溝を作る |
| 三角筋⇔上腕二頭筋 | 三角筋粗面（上腕骨中央1/3）で三角筋が終わり二頭筋の膨隆が始まる |
| 三角筋後部⇔三頭筋長頭 | 三角筋後縁の直下に長頭が続く。Z方向の位置合わせが連続性の核心 |
| 上腕二頭筋⇔腕橈骨筋 | 肘前面で二頭筋腱が腕橈骨筋の起点につながる（**肘窩**の形成） |
| 上腕三頭筋⇔肘頭 | 3頭が肘頭に収束するV字形の溝が特徴的 |
| 前腕屈筋群⇔内側上顆 | 全屈筋群が内側上顆を共通起始とする「扇の要」 |
| 前腕伸筋群⇔外側上顆 | 全伸筋群が外側上顆を共通起始とする「扇の要」 |

### 表面の連続ライン（シルエットの流れ）

**前面ライン**
```
大胸筋（鎖骨部） → 三角筋前部（三角胸筋溝で分離）
  → 上腕二頭筋長頭（三角筋粗面付近で主役交代）
  → 腕橈骨筋（肘外側経由）/ 円回内筋（肘内側経由）
```

**後面ライン**
```
僧帽筋（下部） → 三角筋後部
  → 上腕三頭筋長頭（腋窩後壁から後上腕へ継ぎ目なく接続）
  → 肘頭（3頭の収束点）
  → 総指伸筋（EDC）/ 尺側手根伸筋（ECU）
```

**外側面ライン（最も表面に出る稜線）**
```
三角筋中部（外側膨らみ最大点）
  → 上腕筋（三角筋停止直下、二頭筋外縁から1〜2cm露出）
  → 腕橈骨筋（肘外側から前腕橈側へ）
  → 長橈側手根伸筋（ECRL）
```
- 上腕筋の外側露出が不十分だと三角筋と腕橈骨筋の間に「穴」が生じてバラバラに見える

**内側面ライン**
```
大胸筋（下縁）→ 烏口腕筋
  → 上腕三頭筋内側頭（上腕内側面）
  → 円回内筋 → 尺側手根屈筋（FCU）
```

---

### 視覚的連続性チェックポイント

1. **三角胸筋溝の溝**: 三角筋前部と大胸筋鎖骨部の間に明確な縦溝が見えること
2. **二頭筋の主役交代**: 三角筋粗面付近（X≈0.33〜0.34）で三角筋が途切れ二頭筋の膨隆に移行すること
3. **三頭筋長頭と三角筋後部の継ぎ目**: 後面で三角筋後縁の下に長頭がつながって見えること
4. **肘窩（cubital fossa）の三角形**: 腕橈骨筋（外側縁）・円回内筋（内側縁）・二頭筋腱（中央）で囲まれる凹み
5. **円回内筋の斜走ライン**: 前腕前面を45°で横断し、屈筋群の「顔」になる
6. **前腕の扇形配列**: 屈筋群（前面）と伸筋群（後面）が肘から手首へ扇状に広がること
7. **烏口突起の束**: 烏口腕筋と上腕二頭筋短頭が共に烏口突起から出て平行に走ること

### エコルシェ表面への寄与優先度

**優先度A（必ず見えるべき）**
三角筋（前・中・後）、上腕二頭筋、上腕三頭筋（外側頭・長頭）、腕橈骨筋、円回内筋、総指伸筋（EDC）

**優先度B（見えるべきだが隣接筋に隠れやすい）**
上腕筋（二頭筋外縁から1〜2cm露出）、烏口腕筋（腕外転時に腋窩前壁）、三頭筋内側頭（肘付近のみ）、ECRL/ECRB、FCU/ECU

**優先度C（深層。シルエットへの寄与は間接的）**
浅指屈筋、回外筋、長母指外転筋（ECRBとEDCの間から斜めに出てくる）

### Blender実装上の注意（現在の実装との対応）

- 三角筋3部: 前部y_offset=-0.040、中部=0、後部=+0.040。互いに隣接しつつY位置で区別
- 上腕二頭筋: y_offset=-0.035（長頭）/-0.030（短頭）。三角筋の下から前面に出るよう配置
- 腕橈骨筋: y_offset=-0.015。前腕最外側・橈側に位置し上腕二頭筋から肘外側を経由
- 総指伸筋: y_offset=-0.015（修正済み）。前腕後面中央に位置
- 三頭筋長頭と三角筋後部の後面接続は **y_offset値の一致**（どちらも+0.04付近）で達成
- 腕橈骨筋が「橋渡し」の役割：上腕骨遠位外側1/3まで起始が及ぶため、前腕筋群より近位まで筋腹が来る
- 女性の上腕三頭筋内側頭: Distortionを他頭より低めにすると丸みが出て女性らしい（`male_to_female_differences.md`参照）
