# モデル検証チェックリスト

## 使用方法
1. Blenderで該当部位を正面・背面・側面から撮影
2. Sampleフォルダのリファレンス画像と並べて比較
3. 各項目をチェック（OK / 要修正 / 未確認）

---

## 正面チェック

### 腹部
- [ ] 腹直筋の繊維が垂直か
- [ ] 腱画が適切な位置（臍付近）にあるか
- [ ] 外腹斜筋が斜め下内側に走っているか
- [ ] 前鋸筋の鋸歯状突起が横方向に並ぶか
- [ ] 腹直筋と外腹斜筋の境界（腹直筋鞘）が見えるか

### 胸部
- [ ] 大胸筋が扇状に広がっているか
- [ ] 鎖骨部と胸肋部で繊維方向が異なるか

### 上腕（前面）
- [ ] 上腕二頭筋の繊維が垂直か
- [ ] 2頭の合流が表現されているか

### 大腿（前面）
- [ ] 大腿直筋が中央を垂直に走るか
- [ ] 外側広筋が外側に見えるか
- [ ] 内側広筋が膝内上方で膨隆しているか
- [ ] 縫工筋が斜めに横断しているか

---

## 背面チェック

### 僧帽筋
- [ ] 上部が斜め下外側か
- [ ] 中部が水平か
- [ ] 下部が斜め上外側か

### 広背筋
- [ ] 上部がほぼ水平か
- [ ] 下部がほぼ垂直か
- [ ] 腋窩に向かって収束しているか

### 臀部
- [ ] 大臀筋の繊維が斜め下外側（45°）か
- [ ] 中臀筋が上外側から見えるか

### 大腿（後面）
- [ ] 大腿二頭筋が外側か
- [ ] 半腱・半膜様筋が内側か
- [ ] 3筋が識別できるか

---

## 側面チェック

### 三角筋
- [ ] 前・中・後部が識別できるか
- [ ] 中部が垂直に走るか

### 大臀筋
- [ ] 後方への突出が適切か

### 腓腹筋
- [ ] 2頭（内外側）のダイヤ型が見えるか

## 女性モデル固有チェック（3DAnatomyManから変換時）

### 骨格・プロポーション
- [ ] 骨盤エリア（大殿筋・中殿筋）が肩幅以上の横幅になっているか
- [ ] 肩（三角筋・僧帽筋）が男性より小さく・なで肩方向になっているか
- [ ] ウエストのくびれが明確か（腰幅より細い）

### 胸部
- [ ] 大胸筋のボリュームが男性比 85〜90%程度に抑えられているか
- [ ] エコルシェとして見せる場合、乳房ジオメトリが別オブジェクトとして存在するか（任意）

### 筋肉凹凸の強度
- [ ] 腹直筋の腱画凹凸が男性より控えめか（Displace強度 60%以下）
- [ ] 大腿四頭筋の筋間境界が男性より穏やかか
- [ ] 背中（脊柱起立筋）の溝・隆起が男性より丸みを帯びているか

### 臀部・大腿
- [ ] 大殿筋が丸く後方・側方に突出しているか（男性より丸い）
- [ ] 大腿内側（内転筋群）にある程度のボリュームがあるか
