# 三角筋 — 隣接筋肉との接続・境界

## 接続筋肉マップ（方向別）

| 方向 | 隣接筋肉 | 接続種別 |
|---|---|---|
| 上・後 | **僧帽筋中部**（肩甲棘水平部） | 同一骨起始・直接隣接 |
| 上・前 | **僧帽筋上部** | 肩峰を挟んだ隣接 |
| 前 | **大胸筋（鎖骨部）** | 三角胸筋溝で分離 |
| 前・上 | **広頚筋** | 三角筋筋膜上を薄く覆う（表層） |
| 後・下 | **上腕三頭筋長頭** | 後縁直下に連続 |
| 後・下 | **上腕三頭筋外側頭** | 後縁深層（三角筋粗面付近まで深部） |
| 下・前 | **上腕二頭筋（長頭・短頭）** | 三角筋粗面で主役交代 |
| 深層 | **棘上筋・棘下筋・小円筋** | 三角筋全体の直下（ローテーターカフ） |
| 深層・前 | **烏口腕筋** | 前部三角筋の深層・腋窩前壁 |

---

## 各境界の詳細

### 1. 三角筋後部 ⇔ 僧帽筋中部（最重要・修正対象）

**骨の共有:**
- 僧帽筋中部: 肩甲棘の**上縁（superior lip）**に停止
- 三角筋後部: 肩甲棘の**下縁（inferior lip / crest）**から起始
- → **同一の肩甲棘を挟んで上下に分かれて付着**

**視覚的境界:**
- 肩甲棘の稜線が溝として表面から見える
- 僧帽筋中部（水平）が内側から来て肩甲棘に達し、そのまま三角筋後部が外側へ続く
- 後面から見ると: `僧帽筋中部 → 肩甲棘溝（細い縦溝） → 三角筋後部` の連続ライン

**Blender配置上の要件:**
- 三角筋後部の内側端（X最小値）が肩甲棘位置（X≈0.08〜0.10）に来ること
- 僧帽筋中部の外側端（X最大値）と三角筋後部の内側端が**0〜5mm以内**で接すること
- Y方向: 三角筋後部の Y_max と僧帽筋中部の Y_max がほぼ一致（どちらも後面）

---

### 2. 三角筋前部 ⇔ 大胸筋（鎖骨部）

**三角胸筋溝（deltopectoral groove）:**
- 大頭静脈（cephalic vein）が通る縦溝
- 三角筋前部と大胸筋鎖骨部は「ほぼ連続した筋塊になりかけている」が静脈溝で分離
- 溝は鎖骨下から腋窩へ続く**縦ライン**

**視覚的境界:**
- 前面から: 鎖骨の外側1/3から腋窩に向かう縦の凹み
- 溝の位置: X≈0.10〜0.14（鎖骨付近）
- 三角筋前部の前面Y < 大胸筋鎖骨部の前面Y（三角筋の方がわずかに後ろ側に位置）

---

### 3. 三角筋後部 ⇔ 上腕三頭筋（長頭・外側頭）

**解剖学的関係:**
- 三頭筋**長頭**の近位部は三角筋後部の後縁に覆われる
- 三頭筋**外側頭**の近位端は三角筋の深層にある
- 長頭は三角筋後縁と平行・内側に触知できる隆起として連続

**視覚的ライン（後面）:**
```
三角筋後部（外側） → 三角筋後縁の溝 → 三頭筋長頭（内側）
                              ↑
                    ここに縦溝が現れる
```
- 三角筋粗面（Z≈1.30〜1.35）より下で三頭筋が主役になる
- 三角筋後縁と三頭筋長頭の間の縦溝は後面で目立つランドマーク

**Blender配置上の要件:**
- 三角筋後部の下端（Z_min）と三頭筋長頭の上端（Z_max）がオーバーラップ（3〜5cm）すること
- 三角筋後部 y_offset=+0.040 と 三頭筋長頭 y_offset=+0.040 が一致 → 連続して見える

---

### 4. 三角筋 ⇔ 広頚筋（Platysma）

**解剖学的関係:**
- 広頚筋は**三角筋・大胸筋の筋膜の上**から起始し、鎖骨を越えて頸部へ上行
- 三角筋の表層に薄く重なる（皮下組織内に位置）
- 三角筋そのものとは直接の筋-筋接触ではなく、**筋膜を通じた接続**

**エコルシェ表現上の位置づけ:**
- エコルシェでは広頚筋は「皮膚直下の薄筋」として別扱い
- 三角筋の前縁〜鎖骨上面にかけて広頚筋が薄く覆っている表現が必要な場合、別オブジェクト
- 三角筋配置修正の優先度: 広頚筋との関係は**低優先**（エコルシェでは省略可）

---

### 5. 三角筋 ⇔ 上腕二頭筋（主役交代ライン）

**位置:**
- 三角筋粗面（deltoid tuberosity）= 上腕骨中央1/3（Z≈1.30〜1.37）
- この高さで三角筋が終わり、上腕二頭筋の膨隆が始まる

**視覚的ライン:**
- 前面: 三角筋の下端でくびれが生じ、二頭筋の丸みが現れる
- ここの境界が曖昧だと「上腕が膨らみ続ける」不自然な形になる

---

## 現在の実装値（female_muscle_utils.py基準）

| 筋肉 | anchor | y_offset | 備考 |
|---|---|---|---|
| 三角筋前部 | SHOULDER | -0.040 | 前面 |
| 三角筋中部 | SHOULDER | 0.000 | 中央 |
| 三角筋後部 | SHOULDER | +0.040 | 後面 |
| 三頭筋長頭 | SHOULDER | +0.040 | 後面（三角筋後部と同値） |
| 三頭筋外側頭 | SHOULDER | +0.035 | 後面 |
| 上腕二頭筋長頭 | SHOULDER | -0.035 | 前面 |
| 僧帽筋中部 | 体幹 create_muscle | — | y_offset方式ではなくBODY_Y_OFFSETで配置 |

## 修正チェックリスト（三角筋再配置時）

```
□ 後部内側端（X_min）が肩甲棘に接続（X≈0.08~0.10）
□ 僧帽筋中部の外側端と三角筋後部の内側端の隙間 < 5mm
□ 後部 y_offset が三頭筋長頭 y_offset と一致（どちらも+0.040）
□ 前部の三角胸筋溝が前面から視認できる縦溝になっている
□ 下端（Z_min）で三頭筋長頭とオーバーラップ 3~5cm
□ 三角筋粗面付近（Z≈1.32）で上腕二頭筋への主役交代が自然
```
