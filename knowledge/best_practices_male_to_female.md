## 確立済みワークフロー（大殿筋で検証）

### Step 1: インポートと全トランスフォーム適用

```python
bpy.ops.wm.append(
    filepath=filepath + '/Object/' + src_name,
    filename=src_name,
    directory=filepath + '/Object/'
)
bpy.ops.object.transform_apply(location=True, rotation=True, scale=True)
# → 3DA 右側筋肉は -X（女性左側）に展開される
```

### Step 2: 右側（+X）へのミラー

```python
obj.scale.x = -1.0
bpy.ops.object.transform_apply(scale=True)
bpy.ops.object.mode_set(mode='EDIT')
bpy.ops.mesh.select_all(action='SELECT')
bpy.ops.mesh.flip_normals()
bpy.ops.object.mode_set(mode='OBJECT')
```

### Step 3: 女性化スケール（部位別係数を適用）

```python
obj.scale.x *= FEMALE_X_SCALE
obj.scale.y *= FEMALE_Y_SCALE
bpy.ops.object.transform_apply(scale=True)
```

### Step 4: クリムゾンマテリアル適用

→ `apply_crimson(obj, fiber_angle_deg)` を呼ぶ（female_crimson.pyに関数定義）

### Step 5: 検証（3ステップ）

1. **フィッティング確認**: Ch36(alpha=0.1)と重ねてはみ出し確認
2. **プロポーション確認**: X幅が Ch36 の対応部位と一致しているか数値確認
3. **リファレンス比較**: `Sample/スクリーンショット 2026-03-28 110420.png` と同アングルで目視比較

## 部位別テクスチャ設定（Distortion値）

女性は皮下脂肪が男性より厚いため、部位によって繊維テクスチャの強度を変える。

| 部位 | Distortion | 理由 |
|---|---|---|
| 大殿筋 | **0.05** | 臀部は皮下脂肪厚く繊維ほぼ不可視 |
| 中殿筋 | 0.25 | やや見える |
| 大胸筋 | 0.40 | 乳房下のエコルシェでは見える |
| 腹直筋 | 0.30 | 控えめ（男性の60%） |
| 三角筋 | 0.50 | 皮下脂肪薄い、繊維見える |
| 僧帽筋 | 0.35 | 控えめ |
| 大腿四頭筋 | 0.35 | 控えめ（男性の60%） |
| 上腕二頭筋 | 0.45 | やや見える |

## ビュー設定（重要）

```python
# ✅ 正しい正面ビュー（頭が上）
r3d.view_rotation = Quaternion((1,0,0), math.radians(90))   # qx90 のみ

# ❌ 誤り: qx90 @ q180z → Zが下向き（逆さま）になる

# ✅ 正しい後面ビュー（頭が上）← 2026-04-01 確認済み
qx = Quaternion((1,0,0), math.radians(90))
qz = Quaternion((0,0,1), math.radians(180))
r3d.view_rotation = qz @ qx   # ← 順序が重要: qz を先に適用
```

`set_view('back')` または `set_view('hip_back')` で正しい後面ビューが使えるようになった（`qz_first=True` フラグ）。

## よくある落とし穴

1. **bound_boxのキャッシュ問題**: `obj.bound_box` は bmesh変更後に古い値を返す
   → 実際の頂点位置は `obj.data.vertices` + `view_layer.update()` で取得

2. **scale.x = -1 でのミラー後**: 必ず `flip_normals` を実行しないと裏面表示になる

3. **bmeshで直接頂点スケールを変えようとすると**: 重心計算が複雑になるため
   → `obj.scale.x *= factor` + `transform_apply(scale=True)` の方が確実

4. **3DA 左側筋肉が残っていた場合**: `左大胸筋` 等がインポートされ重複することがある
   → `FEMF_`/`F_` 以外で `左〜` のオブジェクトは非表示化する
