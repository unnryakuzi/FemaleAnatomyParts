# 羽状角・繊維走行まとめ

## 羽状構造の種類

| 種類 | 説明 | 代表筋 |
|---|---|---|
| 紡錘状 (Fusiform) | 繊維が長軸と平行 | 上腕二頭筋・縫工筋 |
| 一羽状 (Unipennate) | 繊維が腱の片側に斜め | 半膜様筋・長母指伸筋 |
| 二羽状 (Bipennate) | 繊維が中央腱の両側 | 大腿直筋・三角筋中部 |
| 多羽状 (Multipennate) | 複数の羽状が集合 | 三角筋中部・棘上筋 |
| 半羽状 (Semipennate) | 一方向のみ斜め | 外側広筋 |

## 主要筋の羽状角（表面から見た繊維の角度）

| 筋名 | 羽状角（長軸に対して） | 走行方向 |
|---|---|---|
| 腹直筋 | 0°（平行） | 垂直 |
| 外腹斜筋 | 45° | 斜め下内側（右側は「＼」） |
| 内腹斜筋 | 45° | 斜め下外側（右側は「／」） |
| 大胸筋（上部） | 30° | 斜め下外側 |
| 大胸筋（中部） | 0° | 水平 |
| 僧帽筋（上部） | 30° | 斜め下外側 |
| 僧帽筋（中部） | 0° | 水平 |
| 僧帽筋（下部） | 30° | 斜め上外側 |
| 広背筋（上部） | 0° | ほぼ水平 |
| 広背筋（下部） | 80° | ほぼ垂直 |
| 三角筋（前部） | 20° | 後下方 |
| 三角筋（中部） | 多羽状 | 垂直 |
| 三角筋（後部） | 20° | 前下方 |
| 大腿直筋 | 10° | ほぼ垂直 |
| 外側広筋 | 15〜20° | 斜め下内側 |
| 内側広筋（下部） | 25〜40° | ほぼ水平 |
| 大臀筋 | 45° | 斜め下外側 |
| 腓腹筋 | 15° | 垂直 |

## 3Dモデルでの判定基準

- **±15°以内** → OK（許容範囲）
- **±15°〜30°** → 要検討（部位によって許容）
- **±30°以上** → 要修正

## 重要な走行の特徴

1. **ほぼ垂直（縦）**: 腹直筋・上腕二頭筋・三頭筋・ハムストリングス・腓腹筋
2. **明確に斜め（45°前後）**: 外腹斜筋・大臀筋・縫工筋
3. **水平**: 僧帽筋中部・広背筋上部
4. **方向が変わる（上下で異なる）**: 広背筋・大胸筋・僧帽筋
5. **ねじれを伴う**: 広背筋（停止部）・大胸筋（停止部）
